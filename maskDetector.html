Machine Learning OpenCV: Mask Detector

Demo of working mask detector

With the COVID-19 global pandemic on our minds, my lab partner (Paul Schellenberg) and I decided to train a machine learning model to detect a face mask. Our model uses python-based OpenCV and TensorFlow to create predictions based on the input video stream, as shown above.

Example Images of Training Set: with_mask & without_mask

The model pulls on 1374 images sectioned into two categories: with_mask or without_mask. Then, the images in each category are split into a training set and a testing set at a ratio of 80:20 (training:testing). The training set is used to generate the prediction model based on a Convolutional Neural Network running 20 Epochs of training.

As the model training progresses through each Epoch, the weights are updated to 'select' the appropriate filter values. This means that as the number of Epochs increases, the model will increase in accuracy (most of the time). Another important part of our model is the pooling layers. Essentially, pooling downsamples the given input so that the model can more quickly analyze the image structure. This means that during the model implementation, it will be computationally less expensive.
