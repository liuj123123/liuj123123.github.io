LabView Assembly Line

Part Detection (metal AND plastic detected = assembled) and Part Rejection (reject only-metal pieces)

In MSE310 (Sensors and Actuators), the final project focused on developing LabView code to control the mini assembly line made by Industrial Control Technology (ICT). National Instrument's (NI) DAQ USD-6501 was used to collect sensor data and NI-MAX was used for the graphical user interface (GUI).

List of sensors: IR sensor (x 3), inductive sensor (x 2), capacitive sensor (x 1)

List of actuators: Rotary DC motor (x 2), rotary solenoid (x 1), linear solenoid (x 2)

Industrial Control Technology - Assembly Line

The main objective was to assemble the plastic ring onto the metal peg. Because the sensor data was digital, we could use boolean logic to detect parameters of the assembly line, such as: material of part, location, quantity, fully assembled, and not assembled.
